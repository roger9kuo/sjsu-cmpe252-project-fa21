Guides to RASA NLU

Project build in 
- Windows 10
- Python 3.8.1

Run venv first
- Command Prompt:> .\venv\Scripts\activate
- Bash:$ source .\venv\Scripts\activate

Create training models before running/testing
- > rasa train

Run the server
- > rasa run actions

In a separate(second) shell, run the program
- > rasa shell
